An interactive exploration of how a single layer perceptron decides to "fire". To operate, we set some defaults for you. Feel free to change Truth mode, pick a row from the truth table, play with weights and threshold. We use red and blue to represent boolean values of 1-True and 0-False respectively. Orange is used for the intermediate steps between taking inputs and producing an output.

Remember: Perceptron is considered successful if it can correctly classify output for all four truth pairs with the same input parameters (weights and threshold). Otherwise, it's not successful.
